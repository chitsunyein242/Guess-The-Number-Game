🔢 Guess the Number Game
 
A fun and interactive number-guessing game built with HTML, CSS, and JavaScript.
The computer chooses a random number between 1 and 50, and your job is to guess it within a limited number of chances.

🚀 Features

•    🎯 Random number generated between 1–50

•    📝 Input box to type your guess

•    🔁 Restart button to start a new game anytime

•    📉 Limited number of chances 

•    📋 Shows all guessed numbers

•    📢 Game messages (too high, too low, correct)

•    🎨 Clean and modern UI 

🛠️ Technologies Used

•    HTML5

•    CSS3

•    JavaScript (ES6)
